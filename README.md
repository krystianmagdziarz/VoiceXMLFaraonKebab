# VoiceXMLFaraonKebab
Projekt realizowany w ramach nauki VoiceXML.
W projekcie zastosowano przemienną inicjatywę jako element wzbogacający dialog, przeznaczony dla doświadczonego użytkownika.
Gramatyka oparta o model SRGS.
